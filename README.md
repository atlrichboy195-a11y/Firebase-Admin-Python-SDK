# Firebase Admin Python SDK

A Python application using the Firebase Admin SDK for server-side operations.

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Download your Firebase service account credentials from the Firebase Console

3. Place your `serviceAccountKey.json` in the project root directory

4. Run the application:
   ```bash
   python main.py
   ```

## Features

- Firestore database operations
- Cloud Storage access
- User authentication management
- Real-time database listeners

## Documentation

For more information, visit the [Firebase Admin Python SDK documentation](https://firebase.google.com/docs/database/admin/start)
